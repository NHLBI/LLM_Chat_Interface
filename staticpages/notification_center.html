What's New

New Model: o3-mini Enabled

The OpenAI o3-mini model is now available in the model dropdown selector. With training data ending October, 2023 and a 100,000 token context window, o3-mini delivers high accuracy and contextual awareness, making it a powerful option for a wide range of applications.

Improved LaTeX Handling

NHLBI Chat now properly handles LaTeX formatting. This enhancement ensures that mathematical expressions, scientific formulas, and other technical notations render correctly.

GPT-3.5 Disabled

The GPT-3.5 model is no longer available in NHLBI Chat. Please use any of the other models available in the model selector.
