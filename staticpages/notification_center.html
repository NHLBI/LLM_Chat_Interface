What's New

Workflows

NHLBI Chat allows you create Workflows, custom prompts designed for a group of users to reliably use the same prompt for a specific task. If you have an idea for a standard prompt to deploy, please send an email to NHLBI_AIWorkflows@nhlbi.nih.gov to begin the process.

Expanded Document Uploads

NHLBI Chat now allows you to upload ten documents per chat. Uploaded documents appear under the chat's title in the chat list and can be deleted there.

New Models

NHLBI Chat now supports o3 (the latest reasoning model), o4-mini (the fastest reasoning model), and GPT-4.1 (technical model excelling in instruction following and coding).

CSV and XLSX Upload

NHLBI Chat now allows you to upload csv and xlsx document types.
